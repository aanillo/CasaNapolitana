## Descripción

**La Casa Napolitana** es una simulación de página web para un restaurante italiano tradicional. El objetivo del proyecto es recrear una experiencia realista de la web de un restaurante donde los usuarios puedan informarse sobre el local, consultar la carta y realizar reservas de mesa.

El proyecto ha sido desarrollado utilizando **Angular**, aplicando una estructura modular y componentes reutilizables para construir una interfaz moderna, dinámica y fácil de navegar.

Este proyecto tiene un enfoque principalmente **educativo y demostrativo**, orientado a practicar el desarrollo de aplicaciones web con frameworks modernos.

---


## Funcionalidades

La aplicación incluye varias secciones típicas de la web de un restaurante:

- **Inicio**  
  Página principal con presentación del restaurante y su estilo gastronómico.

- **Carta / Menú**  
  Visualización de los platos disponibles, con información sobre cada uno de ellos.

- **Reservas**  
  Formulario que permite simular la reserva de una mesa en el restaurante.

- **Información del restaurante**  
  Detalles sobre el local, su filosofía y su propuesta gastronómica.
